## üìñ User Guide

### 1. Preparation
Before starting the program, please prepare the following details from **eStudent**:  
* **Subject Code:** e.g., `ABCT1D04`  
* **Group Number:** e.g., `2001`  
* **Component Codes:** Includes required components such as Lectures, Tutorials, or Labs (e.g., `LTL001`, `T001`)  

---

### 2. Installation & Setup
1. Download the latest `.exe` file from `Releases` in github:
<img width="170" height="130" alt="image" src="https://github.com/user-attachments/assets/88f8395c-7106-4183-ab9e-2125a2f75e4d" />

2. Run **`PolyU_Reg_Helper.exe`**  
3. If blocked by the firewall, please trust and allow the program  
4. In the **Login Credentials** section, enter:  
   * **Student ID**  
   * **Password**  

---

### 3. Adding Subjects
1. Click the **`+ Add`** button to create a new entry in the registration table  
2. Fill in the following fields:  
   * **Code:** Enter the subject code (e.g., `ABCT1D04`)  
   * **Group:** Enter the group number (e.g., `2001`)  
   * **Components:** Enter the component codes  
     > üí° Tip: If there are multiple components, separate them with commas (e.g., `LTL001, T001`). Ensure they match exactly what appears in eStudent.  

---

### 4. Execution
1. **Review:** Double-check all codes and group numbers for accuracy  
2. **Run:** Click the **`RUN NOW`** button or the **`START SCHEDULE`** button to run automatically at the scheduled time  
3. **Monitor:** Track progress in the **Logs** window step by step  
4. **Confirm:** Once finished, a success message will appear  
   * Final step: Log in to eStudent manually to verify that your subjects have been successfully registered
