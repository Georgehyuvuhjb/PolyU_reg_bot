# PolyU Registration Helper
A lightweight, automated tool for PolyU eStudent course registration.

Overview
PolyU Registration Helper is an independent desktop application designed to streamline the subject registration process for PolyU students. By automating the repetitive steps of searching, selecting, and confirming subjects, this tool helps users secure their desired course components with higher efficiency and a cleaner interface.

1. Preparation
Collect your Subject Codes (e.g., ABCT1D04).

Collect the Group Number (e.g., 2001).

Identify the Component Codes you need (e.g., LTL001).

2. Setup
Launch PolyU_Reg_Helper.exe.

Enter your Student ID and Password in the "Login Credentials" section.

3. Adding Subjects
Click the "+ Add" button to create a new row in the table.

Code: Enter the subject code (case-insensitive).

Group: Enter the specific group number.

Components: Enter the component IDs separated by commas (e.g., LTL001, T001).

Tip: Ensure these match exactly what appears in the eStudent system.

4. Execution
Review your list for accuracy.

Click "RUN TASK".

Monitor the Logs window for the "Step-by-Step" progress.

Once finished, a success message will pop up. Check your eStudent "Study List" to confirm.
